When Network Autoconf Admin is in the Rights Included column, it grants the right to create and edit the data in the configuration repository (using the nwamcfg(8) command or the nwam-manager GUI) and to apply the changes made to the configuration repository to the system by nwamd(8) daemon (using the nwamadm(8) command or the nwam-manager GUI).

If Network Autoconf Admin is grayed, then you are not entitled to Add or Remove this right.
